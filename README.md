# Youtube to mp4

pycharmで作ったやつ
| ライブラリ            | 用途                  |
| ---------------- | ------------------- |
| `yt-dlp`         | YouTube 動画をダウンロード   |
| `imageio-ffmpeg` | FFmpeg を使った動画マージ・変換 |
| `pyinstaller`    | スクリプトを exe に変換      |

YouTubeリンクからmp4を作成できます。

ダウンロードが完了すると、  
**デスクトップに `mp4DL` フォルダが作成**されます。
